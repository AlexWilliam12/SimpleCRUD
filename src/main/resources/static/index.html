Documentação para utilizar a API

Método GET para listar todos os alunos: http://127.0.0.1:8080/aluno/listar

Método GET para listar um aluno pelo RA: http://127.0.0.1:8080/aluno/{ra}

É necessário passar um código no lugar de {ra} 
            Ex: http://127.0.0.1:8080/aluno/5c10a6796f

Método POST para cadastrar um aluno: http://127.0.0.1:8080/aluno

É necessário passar um JSON no body 
            Ex:
                {
                    "nome" : "Fulano da Silva",
                    "cep" : "01001000",
                    "numero" : 100,
                    "complemento" : "Casa",
                    "notaAdo1" : 8.75,
                    "notaPi" : 7.25
                }

Método PUT para atualizar um aluno: http://127.0.0.1:8080/aluno/{ra}

É necessário passar um JSON no body e também um código no lugar de {ra} 
            Ex:
                http://127.0.0.1:8080/aluno/5c10a6796f
                {
                    "nome" : "Fulano da Silva",
                    "cep" : "01001000",
                    "numero" : 100,
                    "complemento" : "Casa",
                    "notaAdo1" : 8.75,
                    "notaPi" : 7.25
                }

Método DELETE para excluir um aluno: http://127.0.0.1:8080/aluno/{ra}

É necessário passar um código no lugar de {ra} 
            Ex:
                http://127.0.0.1:8080/aluno/5c10a6796f
